# MonetizeStudio Website

A professional, modern website for MonetizeStudio - turning creators' audiences into revenue through courses, info products, and monetization strategies.

## Files Included

1. **index.html** - Main HTML file with all content and structure
2. **styles.css** - All CSS styling (colors, layouts, animations)
3. **script.js** - JavaScript for interactivity (mobile menu, scroll effects, animations)
4. **monetizestudio-logo.png** - Your logo image
5. **README.md** - This file with instructions

## How to Use

### Option 1: Local Testing
1. Download all files to the same folder
2. Open `index.html` in your web browser
3. All features should work immediately!

### Option 2: Deploy to a Website
1. Upload all files to your web hosting (via FTP, cPanel, or hosting dashboard)
2. Make sure all files are in the same directory
3. Your website will be live at your domain!

### Popular Hosting Options:
- **Netlify** (Free): Drag and drop all files at netlify.com
- **Vercel** (Free): Connect to GitHub or drag files
- **GitHub Pages** (Free): Push files to a GitHub repository
- **Traditional hosting**: Upload via FTP/cPanel

## Customization

### Change Colors
Edit `styles.css` at the top (`:root` section):
```css
--accent-primary: #ff6b35;  /* Main orange color */
--accent-secondary: #f7931e; /* Secondary orange */
```

### Update Content
Edit text directly in `index.html`:
- Find the section you want to change
- Update the text between HTML tags
- Save and refresh

### Change Your Email
Find `hello@monetizestudio.com` in `index.html` and replace with your email

### Update Calendar Link
Find the booking iframe in `index.html` and replace the Google Calendar link

## Need Help?

If you run into any issues:
1. Make sure all files are in the same folder
2. Check that file names match exactly (case-sensitive)
3. Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
